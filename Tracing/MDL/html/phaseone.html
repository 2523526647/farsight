Phase One Help

Parameters

  • Vector Magnitude: When an interpolated vector in the sub-voxel position has magnitude below this value, we take it as a critical point. Typical values range between 0.04 and 0.15.
  • Morph Strength: In graph morphological method with iteration of graph erosion and dilation steps,we set the morphology strength as the number of iterations for graph erosion and dilation. Use higher values to remove longer branches. Typical values range between 30 and 70.
  • Edge Range: If the distance between 2 points is above this distance, we assume there is no edge between them in the graph structure before generating the IW-MST. Use higher values for more connected graphs. Typical values range between 1 and 15.
  • Weight Factor: Used to emphasize the function of either model description or the data description (tradeoff between coverage and conciseness). Use higher values for more connected graphs. Typical values range between 5 and 15.
  • BSpline Order: It is the B-Spline order (= polynominal degree). Note that the current incarnatin of the B-Spline kernal function class is incapabble of producing B-spline of degree >3, thus the possible choise is 0,1 2, 3. The default degree is 3.
  • Number of Levels: If the multilevel approach is used to specifying the number of levels to be greater than one, the number of control points that is specified by the user is the number of control points at the coarsest level of the of the multilevel scheme. The number of control points used at each subsequent level is such that the B-Spline grid size doubke in resolution over the previous level. Hence, the number of levels is the parameter to adjust the control points. Use lower values to get more smooth backbone. Typical values range between 4 and 8.

 

Steps

  • Integratedskel: Detect the ridges and valleys from a 3D vector field
  • BackboneExtract: Extract dendrite backbone from a list of 3D points.
  • SpineExtraction: Extract Spines  from a list of 3D points.
  • BSplineFitting: Compute and Segment-wise  smooth  backbones using  B-spline fitting.
  • RefiningSkeleton: Form streamlines from saddle points and seed points.
